Быстрый старт

Перед отправкой запросов обязательно добавьте ваш IP-адрес в белый список через Telegram-бота (меню бота).
  1. 📌 Получите Authorization TOKEN в боте
  2. 🌐 Добавьте IP в white list в боте
  3. 📨 Отправьте POST-запрос на /linkcreate для генерации ссылки или на /sendemail для отправки письма
Базовый URL: https://api.flytmbot.top

Эндпоинт: Создание ссылки

Тип данных:application/json
Авторизация:Header Authorization: YOUR_TOKEN

Возвращаемые ошибки

Все ошибки приходят в формате JSON с полями action и desc. Ниже представлен полный список возможных кодов ошибок для всех эндпоинтов.

🔒 Ошибки авторизации и доступа

AUTH_TOKEN_UNDEFINED — отсутствует API ключ в заголовке
API_TOKEN_ERROR — неверный API ключ или он не соответствует аккаунту
API_NOT_CREATED — ключ не создан
API_BLOCKED — доступ заблокирован
UNDEFINED_WHITE_IP_LIST — IP не добавлен в white list
IP_UNDEFINED_WHITE_LIST — конкретный IP не найден

⚠️ Ошибки валидации и общие ошибки

ERROR_VALID_DATA — ошибка в передаваемых данных (отсутствуют или некорректны обязательные параметры)
ERROR_EMAIL — указан невалидный email адрес
ERROR_VALIDATE_URL — невалидный формат ссылки
UNDEFINED_CHATID_WORKER — отсутствует chatID
UNDEFINED_HREF — нет ссылки для парсинга
UNDEFINED_NUMBER_PHONE — нет номера телефона
USER_UNDEFINED — пользователь не найден в боте
TEMPLATE_UNDEFINED — нет шаблона и не переданы необязательные поля
DOMAINS_UNDEFINED — нет доменов для данной площадки

📧 Специфичные ошибки эндпоинта /sendemail

EMAIL_OFF — запрашиваемый почтовый сервис временно не работает
EMAIL_ERROR_ADVERT — объявления с таким idAdvert не существует
EMAIL_ERROR_PLATFORMS — данная площадка не поддерживается для email
EMAIL_ERROR_VALID — почтовый сервис счёл email невалидным
EMAIL_ERROR_FORBID — почтовый сервис ограничил доступ (User access denied)
EMAIL_ERROR_UNAVAILABLE — почтовый сервис временно недоступен или ограничил доступ
EMAIL_ERROR_UNV_OLX — отправка на OLX по данному сервису временно недоступна
EMAIL_ERROR_UNV_DPD — отправка на DPD по данному сервису временно недоступна
EMAIL_ERROR_UNV_INPOST — отправка на InPost по данному сервису временно недоступна

🛑 Ошибки сервера и лимитов

CREATE_HREF_UNAVAILABLE — создание ссылок временно недоступно
ERROR_PARSING — ошибка парсера (удалено/отказ площадки)
NUMBER_PHONE_ALREADY_USE — номер уже был использован
ERROR_SERVER_BOT — внутренняя ошибка сервера
API_STOP — API временно остановлено
EMAIL_ERROR — техническая ошибка в email сервисе

Примеры кода

🔗 /linkcreate — Создание ссылки (Python)

import requests, json

url = "https://api.flytmbot.top/linkcreate"
payload = {
    "chatID": "0000",
    "country": "PL",
    "platforms": "OLX",
    "hrefparse": "https://www.olx.pl/d/oferta/rower-trailowy...",
    "numberphone": "00000"
}
headers = {
    "Authorization": "YOUR_TOKEN",
    "Content-Type": "application/json"
}
response = requests.post(url, json=payload, headers=headers)
print(response.text)

Правила безопасности и лимиты

  • Перед запросом обязательно добавьте IP в white list через телеграм-бота.
  • Максимальная длина ссылки hrefparse / title / img — 150 символов.
  • Номер телефона — не более 20 символов.
  • При отсутствии обязательных полей и шаблона — ошибка TEMPLATE_UNDEFINED.
  • Для отправки почты email проходит строгую проверку регулярным выражением, а объявление должно существовать.
Всегда проверяйте поле action в ответе: "OK" — успех, иначе обработайте ошибку по desc.
